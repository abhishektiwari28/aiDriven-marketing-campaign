# Agentic Marketing Campaign Analytics Platform

An AI-driven co-pilot for digital marketing that moves beyond static dashboards to autonomous optimization.

## Features
- **Agentic AI**: Backend agents (Stubbed) for data analysis and strategy recommendation.
- **Real-time Dashboard**: Monitor revenue, conversions, and ROI.
- **Campaign Management**: Launch, pause, and track multi-channel campaigns.
- **AI Assistant**: Wizard-style campaign creation and actionable insights.
- **Premium UI**: Glassmorphism design with responsive layouts.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Recharts, Lucide React.   
- **Backend**: Python, FastAPI (Mocked for Demo).

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)

### running the Frontend
1. Navigate to `frontend/`:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173).

### Running the Backend
1. Navigate to `backend/`:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the server:
   ```bash
   uvicorn main:app --reload
   ```
4. API docs available at [http://localhost:8000/docs](http://localhost:8000/docs).
